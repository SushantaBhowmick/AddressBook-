Introduction

Welcome to addressbook.


Addresbook is a web based application which contains students & Facultys information. Each contact entry consists of a few standard fields. First off all you need to register of the docoments after that you find anyone information, and also be a security maintain possible here.

Purpose and Built-in

The main purpose of this application is to automatically the complete operation of the information. They need maintain hundreds of thousands of record. Also searching be very faster so they can find required details instantly.
